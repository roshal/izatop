
# NumberCoder

A TypeScript library for encoding numbers into short codes and decoding them back.

## Setup

```sh
pnpm install
```

## Testing

```npx nps t```

```sh
npx -- nps test
```

```sh
npx -- tsx --test source/index.test.ts
```

## Linting

```npx nps l```

```sh
npx -- nps lint
```

```sh
npx -- eslint --cache -- source
```
